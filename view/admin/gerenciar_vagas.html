💡 Informações

As vagas são gerenciadas diretamente no banco de dados (tabela vagas).
Para adicionar ou editar vagas, execute os comandos SQL diretamente no MySQL Workbench.

SQL de exemplo:

-- Adicionar nova vaga
INSERT INTO vagas (titulo, slug, descricao, requisitos, tipo, status) 
VALUES (
    'Nome da Vaga',
    'slug-da-vaga',
    '<h3>Descrição</h3><p>Conteúdo em HTML...</p>',
    '<h3>Requisitos</h3><ul><li>Item 1</li></ul>',
    'CLT',
    'aberta'
);

-- Atualizar status
UPDATE vagas SET status = 'pausada' WHERE id_vaga = 1;

-- Listar vagas
SELECT * FROM vagas ORDER BY data_publicacao DESC;
